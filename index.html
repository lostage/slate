    • Introduction

    All API endpoints are hosted from the base URL

    http://www.simplypersonal.com/api

    Endpoints

    Pieces

    http://www.simplypersonal.com/api/pieces

    Response

    {
    "pieces": [
        {
            "id": "SP_S00SavetheDate",
            "name": "Save the Date",
            "designs": [
                {
                    "id": "SP_Brandy",
                    "name": "Brandy",
                    "images": {
                        "png": "/images/generator/previews/SP_S00SavetheDate|Premium_SP_Brandy106_20171110175341.png",
                        "svg": "/images/generator/previews/SP_S00SavetheDate|Premium_SP_Brandy106_20171110175341.svg"
                    },
                    "fields": [
                        {
                            "id": "Top",
                            "multiline": true,
                            "text": [
                                "DR. AND MRS. CYNTHIA TOWNSEND",
                                "REQUEST THE HONOR OF YOUR PRESENCE",
                                "AT THE MARRIAGE OF THEIR DAUGHTER"
                            ]
                        },
                        { ... }
                    ]
                },
                { ... }
            ]
        },
        { ... }
    ]
}

    This lists all of the available pieces along with the designs attached to them. Passing the optional piece_id parameter will return only that piece.

    GET /pieces

    Parameters

    Parameter Required Description
    piece_id false The ID of the piece to return

    Personalize

    Example HTTP Request

    POST http://www.simplypersonal.com/api/personalize

        $ curl http://www.simplypersonal.com/api/personalize \
        -d piece_id="SP_S00SavetheDate" \
        -d look_id="SP_Brandy" \
        -d primary_color="#efabc3" \
        -d primary_font="Walkway Bold" \
        -d secondary_color="#8adbd5" \
        -d secondary_font="Sverige Script" \
        -d top="DR. AND MRS. CYNTHIA TOWNSEND\nREQUEST THE HONOR OF YOUR PRESENCE\nAT THE MARRIAGE OF THEIR DAUGHTER" \
        -d ...

    Response

    {
    "images": {
        "png": "/images/generator/previews/SP_SP_S00SavetheDate|Premium_SP_Brandy106_20171110175341.png",
        "svg": "/images/generator/previews/SP_SP_S00SavetheDate|Premium_SP_Brandy106_20171110175341.svg"
    }
}

    This is used to generate the personalized images after the customer submits their customized fonts, colors and text.

    POST /personalize

    Parameters

    Parameter Required Description
    piece_id true The ID of the piece to personalize
    look_id true
    primary_color true
    primary_font true
    secondary_color true
    secondary_font true
    {{ variable name }} true List all variables and their intended value
